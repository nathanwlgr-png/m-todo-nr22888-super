# Instruções para atualizar index.html com suporte PWA/Mobile

Substitua a seção <head> em index.html com:

<head>
    <meta charset="UTF-8" />
    <link rel="icon" type="image/svg+xml" href="https://base44.com/logo_v2.svg" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0, viewport-fit=cover" />
    <meta name="apple-mobile-web-app-capable" content="yes" />
    <meta name="apple-mobile-web-app-status-bar-style" content="black-translucent" />
    <meta name="apple-mobile-web-app-title" content="Seamaty NR22888" />
    <meta name="theme-color" content="#1e293b" />
    <link rel="manifest" href="/manifest.json" />
    <link rel="apple-touch-icon" href="/icon-192x192.png" />
    <title>Seamaty NR22888 — CRM Veterinário Premium</title>
</head>
