## 🔒 MODO VERDADE ABSOLUTA

### Regras de Ouro
1. **Nunca inventar** — telefone, endereço, CNPJ, Instagram, dados
2. **Nunca prometer resultado falso** — "crescimento 300%" sem base
3. **Nunca usar manipulação antiética** — "apenas hoje" (mentira)
4. **Sempre registrar fonte** — cada dado tem origem
5. **Respeitar LGPD** — nenhum dado privado coletado

### Classificação de Dados

```
🟢 CONFIRMADO
   ├─ CRM (banco interno)
   ├─ Público validado (Google, Instagram)
   └─ Confirmado em ligação/visita

🟡 PROVÁVEL
   ├─ Investigado mas não 100%
   ├─ Lógica clara (clínica X tem equipamento Y)
   └─ Múltiplas fontes concordam

🔴 HIPÓTESE
   ├─ Suposição inteligente
   ├─ Sem validação direta
   └─ "Pode estar correto, mas..."

⚫ NÃO ENCONTRADO
   ├─ Dados não localizados
   ├─ Falta informação
   └─ Marcar como "A validar"
```

### Exemplo: Equipamento

```
NUNCA:
"Sua clínica tem VG2" (sem confirmar)

SIM:
"Confirmado: VG2 (CRM)"
"Provável: VG2 (Google + Instagram)"
"Hipótese: Pode ter VG2 (tamanho clínica)"
"Não encontrado: A validar em visita"
```
