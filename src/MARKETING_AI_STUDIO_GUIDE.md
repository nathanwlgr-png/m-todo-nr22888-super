### Tipos de Campanha

- Foco: Saúde preventiva, exame anual
- Gatilho: Segurança animal
- Resultado: Post + story + email

**⚠️ Emergência/UTI**
- Foco: Paciente crítico, diagnóstico imediato
- Gatilho: Prevenção de morte
- Resultado: Anúncio com urgência

**🔬 Laboratório Completo**
- Foco: Exames bioquímicos, hemogasometria
- Gatilho: Confiança na precisão
- Resultado: Conteúdo educativo

**📊 Hemogasometria**
- Foco: Análise gases sanguíneos (UTI)
- Gatilho: Velocidade = vida
- Resultado: Reel técnico

**🧬 Imunofluorescência**
- Foco: Diagnóstico doenças específicas
- Gatilho: Especialização
- Resultado: Post autoridade

**🔄 Campanha Recorrência**
- Foco: Trazer cliente de volta (exame rotina)
- Gatilho: Pertencimento
- Resultado: Anúncio com CTA

### Incluir Equipamento?
```
SIM: Se campanha educativa termina "coincidentemente"
com equipamento que faz isso

❌ Não: "Use nosso VG2!" (invasivo)
✅ Sim: "Precisa hemogasometria. [..equipamento..]"
```


## 🔒 MODO VERDADE ABSOLUTA — CRÍTICO

### Regras Invioláveis
1. **Nunca inventar dados** — telefone, enderço, CNPJ, resultados
2. **Nunca prometer resultado falso** — "crescimento 300%" sem base
3. **Nunca manipulação antiética** — "apenas hoje" (mentira)
4. **Sempre registrar fonte** — cada dado tem origem
5. **Respeitar LGPD** — nenhum dado privado coletado

### Separar Claramente
```
FATO: "Equipamento faz 26 parâmetros" (confirmado)
OPINIÃO: "É o melhor do mercado" (subjetivo)
HIPÓTESE: "Seu cliente aprovaria" (pode estar errado)

SEMPRE MARCAR DIFERENÇA
```

### Se Não Sabe
```
❌ "Economiza R$1 por exame"
✅ "Pode economizar até R$1 (validar seu volume)"

❌ "Resultado garantido"
✅ "Resultado rápido (em 5 minutos)"

❌ "Cliente ficará feliz"
✅ "Muitos clientes reportam satisfação"
```
